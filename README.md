# tatiana-battistini-portfolio
BATTISTINI.EXE é um portfólio interativo inspirado no Windows 95, desenvolvido com HTML, CSS e JavaScript. O projeto apresenta minha trajetória em Engenharia de Software, habilidades técnicas, projetos e experiências por meio de uma interface retrô, criativa e voltada para recrutadores e profissionais da área de tecnologia.
